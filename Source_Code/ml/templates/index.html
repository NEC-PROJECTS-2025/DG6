About the Project

This project utilizes advanced deep learning techniques to classify waste into categories like plastic, paper, metal, and glass. By automating waste classification, it promotes recycling and sustainable waste management practices.

Designed for individuals and organizations, it ensures high accuracy through advanced algorithms and supports smart city initiatives for environmental sustainability.

The project uses advanced models like InceptionV3 for efficient and accurate image classification. Additional models such as Vgg16, AlexNet, and MobileNetV2 ensure scalability and performance, improving classification reliability for smarter recycling.

How It Works

  • Upload

    The user uploads an image of the waste material through the application.

  • Preprocessing

    The uploaded image is resized, normalized, and processed to prepare it for classification.

  • Classification

    The deep learning model identifies the type of waste (e.g., plastic, paper, metal, glass, cardboard) using trained algorithms.

  • Result

    The application displays the classification result and provides recycling or disposal tips for the waste.

Metric Formulas

Accuracy: Accuracy = (TP + TN) / (TP + TN + FP + FN)

Precision: Precision = TP / (TP + FP)

Recall: Recall = TP / (TP + FN)

F1 Score: F1 Score = 2 * (Precision * Recall) / (Precision + Recall)

Project Flowchart

This flowchart represents the workflow of the waste classification system.

1. Data Preprocessing Phase

Random Oversampling: To handle class imbalance in the dataset, additional samples are generated for underrepresented categories.

Data Augmentation: Techniques such as shear transformations and rotation are applied to increase the diversity of the dataset.

2. Data Separation

The dataset is split into three subsets:

  • Train (70%): Used for training the model.
  • Validation (15%): Used to fine-tune and validate the model's hyperparameters.
  • Test (15%): Used for final performance evaluation.

3. Classification Phase

MBWO Optimization-Based Hyperparameter Tuning: Modified Butterfly Optimization (MBWO) is used to optimize hyperparameters for improved model performance.

Optimized InceptionV3: The classification model employed is an InceptionV3 architecture optimized for high accuracy.

4. Classification Output

The system classifies waste into six categories:

  • Cardboard
  • Metal
  • Plastic
  • Glass
  • Paper
  • Trash
